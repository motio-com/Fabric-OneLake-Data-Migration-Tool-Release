Frequently Asked Questions

Do I need Azure/Fabric credentials to use this tool? ▼

Yes, you'll need valid Azure and Microsoft Fabric credentials to authenticate and access your OneLake workspace. The tool prompts you for these on first run and securely stores them for future use.

What file formats are supported? ▼

The tool supports CSV and Excel files (xlsx, xls) with automatic format detection and conversion. Additional formats can be added based on community feedback.

Can I migrate large files? ▼

Yes! The tool is optimized for batch processing and handles large files efficiently. For very large datasets (1GB+), you may want to split them into smaller chunks for optimal performance.

Is my data secure during migration? ▼

Absolutely. The tool uses encrypted HTTPS connections and Azure's secure authentication. Your credentials are never logged or transmitted outside your machine, and all data transfer is encrypted.

What happens if the migration fails? ▼

The tool provides detailed error messages and logs to help you troubleshoot. Failed migrations can be safely retried without losing data, and partial migrations don't corrupt your OneLake workspace.

Is this tool really free? ▼

Yes! This tool is completely free as part of the Motio Labs initiative to support the Fabric and Power BI community. It's our way of giving back to the community.

How do I report bugs or request features? ▼

Please visit our GitHub repository to open an issue. We actively monitor community feedback and incorporate it into our development roadmap. Your input helps make the tool better for everyone.
